Forgeflow V4.3.30 · 28 May 2026

Release: Retention Preview And First-Run Routing

Forgeflow 4.3.30 refines context retention and first-run guidance. It trims duplicate retention noise, adds a read-only cleanup preview, makes next-work ranking policy explicit, and adds wrapper-level smoke coverage for command launch paths.

Context Retention

  • /forgeflow-context-retention now reports broad context separately from nested latest and agent-packet buckets so advisory size and stale signals are less repetitive.
  • New --preview-cleanup output lists manual stale-file and history-trim candidates without deleting, archiving, compacting, refreshing, or mutating artifacts.
  • Preview output dedupes nested bucket paths before rendering, with regression coverage to keep the manual cleanup list trustworthy.

Next-Work Guidance

  • Next-work ranking now uses an explicit source-priority policy so future signal families have a clear place to be ranked.
  • First-run fallback guidance now starts with install health, project orientation, code map, and trends before implementation when no stronger signal exists.
  • Project-intelligence tests cover the ranking policy and first-run fallback path.

Command And Release Safety

  • New command-wrapper smoke coverage verifies the context-retention and post-release install verifier wrappers resolve executable repo-local helpers and invoke them with safe arguments.
  • Command coverage now catches helpers probed with -x when the source helper is not executable.
  • Existing helpers used by executable probes now carry executable mode in the source tree.
  • The wrapper smoke is wired into release-check, release gate docs, release process docs, and release-version coverage.

Validation

  • Full scripts/forgeflow/test-*.js suite
  • node scripts/forgeflow/smoke-check.js --mode source --json
  • git diff --check
  • Focused context-retention, project-intelligence, command-coverage, command-wrapper, docs, install, update, runtime-helper, and release-version tests
  • Forgeflow review approval from Smith, Warden, Lumen, and Atlas
