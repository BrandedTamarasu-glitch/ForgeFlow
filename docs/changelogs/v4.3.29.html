Forgeflow V4.3.29 · 28 May 2026

Release: Cleaner Guidance And Context Retention

Forgeflow 4.3.29 sharpens the learning and release loops. It keeps first-run projects from chasing missing failure digests, makes next-work recommendations prefer current actionable evidence, adds explicit profile confirmation prompts, and introduces a read-only context-retention check for local guidance artifacts.

Project Intelligence And Next Work

  • First-run missing failure digests now stay informational until Forgeflow has captured a real failed command.
  • Next-work ranking now prioritizes current actionable risks before readiness, history-only, or advisory calibration signals.
  • Regression coverage protects the first-run failure-digest and ranking behavior in the project-intelligence tests.

Profile And Context Guidance

  • /forgeflow-profile-review now renders explicit confirmation prompts before recording, moving, or superseding user preferences.
  • New /forgeflow-context-retention reviews local context freshness, broad context size, code-map history, and context-advisor history without deleting or mutating artifacts.
  • The new retention helper is installed as a managed runtime helper and covered by command, health, release, update, and manifest checks.

Release Verification

  • /forgeflow-post-release-install-verify now treats mode-only install drift as informational instead of repair-required.
  • Release and wiki docs now include the context-retention test so future release checks keep the new helper covered.
  • A review-found executable-bit regression on the new helper was fixed and guarded by install-manifest coverage.

Validation

  • Full scripts/forgeflow/test-*.js suite
  • node scripts/forgeflow/smoke-check.js --mode source --json
  • git diff --check
  • Focused project-intelligence, profile-review, post-release install verify, context-retention, install-manifest, command, docs, update, and release-version tests
  • Forgeflow review approval from Smith, Warden, and Compass after the executable-bit fix
