Forgeflow V4.3.70 · 15 June 2026

Release: Automatic Lean Advisory Activation

Forgeflow 4.3.70 wires lean guidance into the normal review and ship path. The lean lane now runs automatically during review when the helper is available, and ship preparation records lean-readiness evidence for work that selected a lean path.

Review Flow

  • /review now attempts the lean review helper automatically after context assembly.
  • The helper writes compact Markdown and JSON advisory artifacts under the latest context directory.
  • Arbiter and Compass receive the lean advisory as prioritization guidance only; it cannot block approval or override evidence.

Ship Flow

  • /ship checks whether a lean decision exists and warns when the matching lean report or implementation-notes ceiling evidence is missing.
  • ship-prepare.sh writes lean-readiness.json and includes a Lean Readiness section in the prepared PR body.
  • The readiness check is advisory, preserving the existing review, secret-scan, validation, PR, push, and CI gates.

Docs

  • The README and workflow-command wiki now describe the automatic lean advisory lane.
  • The hosted docs entry points to this release as the latest packaged changelog.
