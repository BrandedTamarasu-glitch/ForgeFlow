Forgeflow V4.3.47 · 2 June 2026

Release: Decision Guidance And Update Verification

Forgeflow 4.3.47 tightens the safe workflow surfaces added in 4.3.46 so users get clearer next-work guidance, a cleaner release-consumption finish state, non-recursive learning capture prompts, and more accurate installed-runtime drift explanations.

Project Decision Briefs

  • /forgeflow-project-brief now includes explicit recent changes, avoid-first guidance, validate-first guidance, and high-care files in both JSON and Markdown output.
  • Recent-change fallback now supports the real code-topology artifact shape, including object-map changed_sections, array-shaped sections, changed_files, and changed_work.

Release And Learning Follow-Through

  • /forgeflow-release-consumption-loop now reports a clear release-consumption-complete or attention badge after update verification, downstream smoke, and release-consumption checks.
  • /forgeflow-workflow-ending-capture now shows the matching recorder command in its learning nudge instead of pointing users back to the advisory helper.
  • Watch-mode workflow endings keep the learning nudge read-only and leave the recorder command blank when no capture is required.

Installed Runtime Verification

  • /forgeflow-update-verify now reports the source version alongside the installed version when local Git provenance is available.
  • Drift guidance now separates missing version metadata, source/install drift after a new checkout or release, and runtime drift that still needs repair.
  • Source-version detection supports normal checkouts, detached HEADs, linked Git worktrees, and packed refs.

Docs And Validation

  • README and wiki pages now describe the richer decision brief, release-consumption completion badge, workflow-ending learning nudge, and update-verification drift guidance.
  • Focused tests cover the new topology fallback, recorder nudge behavior, expected and unexpected update drift paths, and linked-worktree source detection.
  • Full scripts/forgeflow/test-*.js suite
  • node scripts/forgeflow/smoke-check.js --mode source --json
  • node scripts/forgeflow/test-doc-links.js
  • git diff --check
  • Forgeflow review approved by Warden, Smith, and Compass after topology-shape, learning-nudge, and linked-worktree findings were fixed.
