Forgeflow V4.3.8 · 21 May 2026

Patch Release: Safe Command Output Reduction

Forgeflow 4.3.8 turns the RTK token-reduction lessons into native, guarded helpers. The new path is allowlist-only and avoids shell-wide interception.

New Helpers

  • /forgeflow-compact-output compacts test, typecheck, lint, log, grep, JSON, status, and tree output when the command class is safe.
  • /forgeflow-failure-digest writes a compact digest with raw-required status, omitted-line counts, and file/line references.
  • /forgeflow-noisy-command suggests narrower invocations before large command output enters context.

Safety Contract

  • Diffs, patches, SHAs, exact file lists, and tool-fed output pass through raw.
  • Malformed parse input returns raw output with an explicit reason.
  • Nonempty input never becomes silent empty output.
  • The synced tooling pattern library now includes the safe/unsafe command-output matrix.
