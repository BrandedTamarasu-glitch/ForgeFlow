Forgeflow V4.3.71 · 15 June 2026

Release: Higher-Precision Lean Findings

Forgeflow 4.3.71 makes the lean review lane more useful and less noisy. Lean findings now explain confidence, replacement options, evidence, and proof steps before a developer acts on them.

Lean Review Precision

  • /forgeflow-lean-review findings now include confidence, replacement guidance, estimated net lines, why-safe evidence, why-not-safe evidence, and proof steps.
  • The helper suppresses obvious semantic false positives, such as locale-aware sort logic, into visible skipped-boundary records instead of noisy findings.
  • Dependency, topology, invocation, and marker evidence remain static and advisory; they do not prove runtime behavior or change review verdicts.

Docs

  • The README now describes precision fields and suppressed candidates in the lean review lane.
  • The wiki command reference and context-intelligence pages were updated for the richer lean finding shape.
  • The hosted docs and wiki home point to this release as the latest packaged changelog.
